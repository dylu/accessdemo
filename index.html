LeagueSearch [In Progress]

[Will update with screenshots, after a bit of polishing!]

LeagueSearch is a project I made for Rainmeter.

This project was inspired by the website LoLNexus, which allows players of the popular game League of Legends to search up information regarding other players.

This project gave me an opportunity to implement something I wanted to have personally, as a quick simple utility without opening a separate webpage (especially if I were in-game or otherwise preoccupied). In addition, I got a chance to play around with Riot Games' API, and use different unfamiliar tools to fit my needs.

This project utilizes Riot Games' API, and follows Rainmeter's own specifications.
For more information, feel free to check out the github project.
